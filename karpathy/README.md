Based on Andrej Karpathy YouTube lectures: https://www.youtube.com/@AndrejKarpathy

- makemore  
  A program that generates new people names based on list of existing names. We start from a simple program using char-based statistics which evolves to a program using multiple layers of neurons trained using backpropagation.
- byte-pair-encoding  
  An algorithm of text compression suitable for language model input.
- gpt  
  Getting familiar to Attention and Transformer, from a program that generates Shakespeare-like texts to an implementation of OpenAI GPT-2 model.
